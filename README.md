# Wolfstein 2D Game 🎮

A 2D raycasting game inspired by the classic Wolfenstein 3D, built from scratch using Python and Pygame.

![Gameplay](screenshots/gameplay.gif)

## 🎯 About
Recreated the Wolfenstein 3D experience in 2D as a solo project. The game features a raycasting engine for pseudo-3D rendering, enemy AI, sprite animation, collision detection, and immersive sound — all implemented from scratch without any game engine.

## ✨ Features
- Raycasting-based 3D rendering engine
- Enemy AI with pathfinding logic
- Sprite and object rendering
- Collision detection system
- Weapon system
- Sound effects
- Custom map design

## 🛠️ Tech Stack
- **Language:** Python
- **Library:** Pygame
- **Concepts:** Raycasting, Game Physics, AI Logic, Sprite Rendering

## 📁 Project Structure
```
├── main.py              # Entry point
├── player.py            # Player movement & controls
├── raycasting.py        # Raycasting engine
├── npc.py               # Enemy AI logic
├── map.py               # Map layout
├── object_renderer.py   # Sprite & texture rendering
├── object_handler.py    # Game object management
├── weapon.py            # Weapon system
├── sound.py             # Sound effects
├── pathfinding.py       # Enemy pathfinding
├── settings.py          # Game configuration
├── resources/           # Game assets
│   ├── textures/        # Wall & floor textures
│   ├── sprites/         # Enemy & object sprites
│   └── sound/           # Background music & SFX
└── screenshots/         # Gameplay screenshots & GIF
```

## 🚀 How to Run

1. Clone the repository
```bash
git clone https://github.com/VinayKumar-B-2005/Wolfstein-2D-Game.git
cd Wolfstein-2D-Game
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Run the game
```bash
python main.py
```
